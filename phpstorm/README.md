## Search

CTRL + SHIFT + A <br />
search actions/ configuration options

CTRL + SHIFT + N <br />
search files <br />
-> usually easier to press shift x2

CTRL + ALT + SHIFT + N <br />
search "symbol" aka "class/methods/etc" within your project -> good for jumping right to framework declaration, or css class name/etc <br />
-> usually easier to press shift x2
